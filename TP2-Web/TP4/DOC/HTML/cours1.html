Apprenez les bases du HTML

Comprenez la structure du web et créez votre première page en quelques minutes.

1. Qu'est-ce que le HTML ?

HTML signifie HyperText Markup Language. Il permet de structurer le contenu d’une page web grâce à des balises.

2. Structure d’une page

<!DOCTYPE html>
<html lang="fr">
  <head>
    <title>Ma page</title>
  </head>
  <body>
    <h1>Bonjour</h1>
  </body>
</html>

3. Balises essentielles

<h1>
Titre principal
<p>
Paragraphe
<a>
Lien
<img>
Image
